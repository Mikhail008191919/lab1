# Лабораторная работа №1

## Тема
Работа с Git и GitHub. Вариант 2b: Функции смены регистра строк.

## Описание
Программа принимает строку и выводит её в верхнем и нижнем регистре.

## Как запустить
g++ program.cpp -o program
./program

## Пример работы
Введите строку: Привет
Верхний регистр: ПРИВЕТ
Нижний регистр: привет
